GenAI Chatbot with Google ADK

Hands-on Kaggle notebook that builds an AI agent using Google’s Agent Development Kit (ADK) and Gemini models, including tool use via Google Search.

Project Overview

This project follows Google’s 5‑day Kaggle Agents course to create a simple AI agent powered by Gemini. The notebook shows how to configure the Gemini API key, import ADK components, define an agent with the google_search tool, and run it with an in‑memory runner to answer questions that require up‑to‑date information.[web:78]

The agent is built in Python using the Google ADK library and runs inside Kaggle Notebooks, with an optional ADK web UI for inspecting traces, sessions, and agent behavior.[web:68][web:78]

Main steps implemented

  • Configure Gemini API key via Kaggle Secrets and environment variables.
  • Import ADK components such as Agent, InMemoryRunner, and google_search tool.[web:78]
  • Define a root agent that uses Gemini to reason and call Google Search when it needs current information.[web:78]
  • Run the agent with run_debug() to answer questions and inspect the reasoning trace.
  • Optionally start the ADK web server and access the dev UI through a proxied URL inside Kaggle.
